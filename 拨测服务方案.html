01 背景与目标
1.1 业务背景

AI API 代理/网关服务作为连接终端用户与上游 AI 能力的核心枢纽，其网络质量直接决定了服务的可用性与用户体验。与传统的 CDN 或 API 网关不同，AI API 代理面临以下独特挑战：

流式
SSE 长连接传输
TLS
透传与证书链
全球
跨境节点调度
  • 长连接稳定性：AI 模型推理响应时间通常 2-30s，且采用 SSE 流式输出，对网络抖动和断连极其敏感
  • TLS 透传需求：代理业务需在用户无感知的前提下完成请求转发，TLS 握手延迟直接影响首字节时间
  • 跨境链路质量：代理用户需要通过海外出口访问上游 AI 服务，跨境链路的丢包、延迟、路由绕行是核心瓶颈
  • 数据飞轮闭环：拨测数据是构建"质量感知→动态调度→体验提升→更多流量→更优数据"飞轮的关键输入

1.3 核心原则

3.2 拨测模式
主动拨测 (Synthetic)

从全球探针节点主动发起测试请求，模拟用户行为。优点：可复现、可对比、不受业务流量影响。缺点：不能完全反映真实用户分布。

定时触发 全局视角
被动采集 (Real User)

从真实用户请求中提取网络质量指标（RUM）。优点：真实体感、无需额外流量。缺点：依赖客户端埋点、无法覆盖无流量区域。

06 测试流程与用例
6.1 标准测试流程
Step 1 — 配置下发
Central Controller 向目标节点 Agent 推送测试任务（目标地址、协议、参数、频次）
Step 2 — 环境准备
Agent 校验网络连通性、清理缓存（DNS 缓存、连接池），确保测试环境干净
Step 3 — 测试执行
按 L1→L4 顺序依次执行，每层测试结果打标签归档；异常时触发即时诊断（Traceroute/MTR）
Step 4 — 数据上报
Agent 将结构化测试数据推送至 Kafka Topic，附带节点信息、时间戳、测试参数元数据
Step 5 — 聚合分析
Flink 实时流计算分位值、成功率等聚合指标，写入 ClickHouse；离线 T+1 生成日报
Step 6 — 告警决策
规则引擎评估聚合指标，触发阈值告警→告警网关→PagerDuty/企微/飞书

8.1 告警分级
级别条件响应时间通知渠道
P0CSR < 90% 或 综合评分 < 605min 内响应电话 + PagerDuty + 企微

P3指标劣化趋势或潜在风险当日处理飞书 + 邮件
8.2 告警降噪策略
  • 连续 N 次触发：连续 3 次拨测异常才触发告警，避免单次抖动误报
  • 影响范围过滤：仅影响单节点的异常降级为 P2，多节点同时异常提升为 P0
  • 维护窗口：计划维护期间暂停相关告警
  • 关联抑制：L1 异常时抑制 L2-L4 的下游告警，避免告警风暴
  • 趋势预判：基于 7 天同环比数据，仅在显著偏离时告警
8.3 Agent 自监控
自监控项方法异常处理
Agent 存活心跳上报（每 10s）连续 3 次心跳丢失 → 节点标记离线
测试任务执行执行日志 + 结果上报连续 5 次无上报 → 节点标记异常
系统资源CPU/内存/磁盘监控CPU > 80% 持续 5min → 降级测试频率
时钟同步NTP 偏差检测偏差 > 100ms → 告警并标记数据不可信
8.4 灾备与降级
  • Controller 多活：Central Controller 部署多实例，主节点故障自动切换
  • Agent 本地缓存：与 Controller 通信中断时，Agent 按本地缓存配置继续执行测试
  • 数据管道降级：Kafka 不可用时本地暂存数据，恢复后批量补传
  • 探针故障转移：单节点离线时自动将测试任务调度至同城备选节点

# ICMP Ping 拨测脚本
# 输出: JSON 格式的 RTT 统计与丢包率
# 用法: ./icmp_ping_test.sh <target> [count] [interval] [timeout]

  RTT_MIN="null"

  RTT_MAX="null"
  RTT_MDEV="null"
fi

# 计算抖动 (Jitter = mdev)
JITTER="$RTT_MDEV"

# 输出 JSON

#=============================================
# TCP Connect 拨测脚本
# 输出: JSON 格式的连接延迟与成功率
# 用法: ./tcp_connect_test.sh <host> <port> [count] [timeout]
#=============================================

set -euo pipefail

HOST="${1:?用法: $0 <host> <port> [count] [timeout]}"
PORT="${2:?用法: $0 <host> <port> [count] [timeout]}"
COUNT="${3:-10}"

TIMESTAMP=$(date -u +"%Y-%m-%dT%H:%M:%SZ")

SUCCESS=0
FAIL=0
LATENCIES=()

for i in $(seq 1 "$COUNT"); do
  START_NS=$(date +%s%N)
  if timeout "$TIMEOUT" bash -c "echo >/dev/tcp/$HOST/$PORT" 2>/dev/null; then
    END_NS=$(date +%s%N)
    ELAPSED_MS=$(( (END_NS - START_NS) / 1000000 ))
    LATENCIES+=("$ELAPSED_MS")
    SUCCESS=$((SUCCESS + 1))
  else
    FAIL=$((FAIL + 1))
  fi
done

# 计算统计
TOTAL=$((SUCCESS + FAIL))
CSR=$(echo "scale=4; $SUCCESS / $TOTAL * 100" | bc 2>/dev/null || echo "0")

if [ ${#LATENCIES[@]} -gt 0 ]; then
  SORTED=($(printf '%s\n' "${LATENCIES[@]}" | sort -n))
  MIN=${SORTED[0]}
  MAX=${SORTED[${#SORTED[@]}-1]}
  AVG=$(echo "${LATENCIES[@]}" | tr ' ' '\n' | awk '{s+=$1;n++} END {printf "%.2f", s/n}')
  # P90
  P90_IDX=$(echo "${#SORTED[@]} * 90 / 100" | bc)
  P90=${SORTED[$P90_IDX]:-$MAX}
  # P99
  P99_IDX=$(echo "${#SORTED[@]} * 99 / 100" | bc)
  P99=${SORTED[$P99_IDX]:-$MAX}
else
  MIN="null"; MAX="null"; AVG="null"; P90="null"; P99="null"
fi

cat <<EOF
{
  "test_type": "tcp_connect",
  "target": "$HOST:$PORT",
  "timestamp": "$TIMESTAMP",
  "params": {
    "count": $COUNT,
    "timeout_sec": $TIMEOUT
  },
  "result": {
    "connect_success": $SUCCESS,
    "connect_fail": $FAIL,
    "connect_success_rate": $CSR,
    "latency_min_ms": $MIN,
    "latency_avg_ms": $AVG,
    "latency_max_ms": $MAX,
    "latency_p90_ms": $P90,
    "latency_p99_ms": $P99
  },
  "status": "$([ "$SUCCESS" -eq 0 ] && echo "fail" || echo "ok")"
}

#!/usr/bin/env bash
#=============================================
# DNS 解析拨测脚本
# 输出: JSON 格式的解析延迟、结果与一致性
# 用法: ./dns_resolve_test.sh <domain> [dns_servers]
#=============================================

set -euo pipefail

DOMAIN="${1:?用法: $0 <domain> [dns_server1,dns_server2,...]}"
DNS_SERVERS="${2:-8.8.8.8,1.1.1.1,223.5.5.5}"
TIMESTAMP=$(date -u +"%Y-%m-%dT%H:%M:%SZ")

# 检查 dig 是否可用
if ! command -v dig &>/dev/null; then
  echo '{"error": "dig 命令不可用，请安装 dnsutils"}' >&2
  exit 1
fi

RESULT_SERVERS=()
ALL_RESULTS=()

for DNS_SERVER in $(echo "$DNS_SERVERS" | tr ',' ' '); do
  START_NS=$(date +%s%N)
  DIG_OUTPUT=$(dig "@$DNS_SERVER" "$DOMAIN" A +noall +answer +stats +dnssec 2>&1) || true
  END_NS=$(date +%s%N)
  QUERY_TIME_MS=$(( (END_NS - START_NS) / 1000000 ))

  # 提取解析结果
  ANSWERS=$(echo "$DIG_OUTPUT" | grep -E "^[^;].*IN\s+A\s+" | awk '{print $NF}' | sort | tr '\n' ',' | sed 's/,$//')

  # 提取 dig 自报的查询时间
  DIG_QUERY_MS=$(echo "$DIG_OUTPUT" | grep "Query time" | grep -oP '\d+' | head -1 || echo "$QUERY_TIME_MS")

  # DNSSEC 状态
  DNSSEC_STATUS=$(echo "$DIG_OUTPUT" | grep -c "RRSIG" || echo "0")
  if [ "$DNSSEC_STATUS" -gt 0 ]; then
    DNSSEC="signed"
  else
    DNSSEC="unsigned_or_not_checked"
  fi

  # 检查是否有 AD flag (Authenticated Data)
  AD_FLAG=$(echo "$DIG_OUTPUT" | grep -c "flags:.*ad" || echo "0")

  RESULT_SERVERS+=("$DNS_SERVER")
  if [ -n "$ANSWERS" ]; then
    ALL_RESULTS+=("$ANSWERS")
  fi

  # 输出每个 DNS 服务器的结果
  echo "  \"$DNS_SERVER\": {" >&3
  echo "    \"query_time_ms\": $DIG_QUERY_MS," >&3
  echo "    \"wall_time_ms\": $QUERY_TIME_MS," >&3
  echo "    \"answers\": \"$ANSWERS\"," >&3
  echo "    \"dnssec\": \"$DNSSEC\"," >&3
  echo "    \"ad_flag\": $AD_FLAG" >&3
  echo "  }" >&3
done 3>/tmp/dns_results.json

# 一致性检查
UNIQUE_RESULTS=($(printf '%s\n' "${ALL_RESULTS[@]}" | sort -u))
if [ ${#UNIQUE_RESULTS[@]} -le 1 ]; then
  CONSISTENCY="consistent"
else
  CONSISTENCY="inconsistent"
fi

# 构建输出
cat <<EOF
{
  "test_type": "dns_resolve",
  "target": "$DOMAIN",
  "timestamp": "$TIMESTAMP",
  "params": {
    "dns_servers": "$(echo $DNS_SERVERS | sed 's/ /,/g')"
  },
  "result": {
    "consistency": "$CONSISTENCY",
    "unique_result_count": ${#UNIQUE_RESULTS[@]},
    "servers": $(cat /tmp/dns_results.json | paste -sd',' | sed 's/^/{/' | sed 's/$/}/')
  },
  "status": "$([ "$CONSISTENCY" = "consistent" ] && echo "ok" || echo "warn")"
}
EOF

rm -f /tmp/dns_results.json

# TLS 握手拨测脚本

# 用法: ./tls_handshake_test.sh <host> [port] [timeout]

# TLS 握手延迟测量

OPENSSL_OUTPUT=$(echo | timeout "$TIMEOUT" openssl s_client -connect "$HOST:$PORT" -servername "$HOST" 2>&1) || true

if echo "$OPENSSL_OUTPUT" | grep -q "TLSv1.3"; then
  PROTOCOL="1.3"

  PROTOCOL="1.2"
fi

# 密码套件
CIPHER=$(echo "$OPENSSL_OUTPUT" | grep "Cipher" | awk '{print $NF}' || echo "unknown")

# 证书信息
CERT_INFO=$(echo "$OPENSSL_OUTPUT" | sed -n '/-----BEGIN CERT/,/-----END CERT/p' | openssl x509 -noout -subject -dates -issuer 2>/dev/null || echo "parse_failed")

CERT_SUBJECT=$(echo "$CERT_INFO" | grep "subject=" | sed 's/subject= //' || echo "unknown")
CERT_ISSUER=$(echo "$CERT_INFO" | grep "issuer=" | sed 's/issuer= //' || echo "unknown")
CERT_NOT_BEFORE=$(echo "$CERT_INFO" | grep "notBefore=" | sed 's/notBefore=//' || echo "unknown")

# 证书过期检查
CERT_EXPIRED="false"
if [ "$CERT_NOT_AFTER" != "unknown" ]; then

  NOW_EPOCH=$(date +%s)
  DAYS_LEFT=$(( (EXPIRY_EPOCH - NOW_EPOCH) / 86400 ))
  if [ "$NOW_EPOCH" -gt "$EXPIRY_EPOCH" ] 2>/dev/null; then
    CERT_EXPIRED="true"
  fi
else
  DAYS_LEFT="unknown"
fi

# 协议降级测试
DOWNGRADE_TLS10="blocked"
DOWNGRADE_TLS11="blocked"
echo | timeout 5 openssl s_client -connect "$HOST:$PORT" -tls1 2>&1 | grep -q "Cipher" && DOWNGRADE_TLS10="vulnerable" || true

HSTS_HEADER=$(curl -sI "https://$HOST" 2>/dev/null | grep -i "strict-transport-security" || echo "not_found")

{

  "timestamp": "$TIMESTAMP",

    "cert_issuer": "$CERT_ISSUER",
    "cert_not_before": "$CERT_NOT_BEFORE",
    "cert_not_after": "$CERT_NOT_AFTER",
    "cert_days_remaining": $DAYS_LEFT,

E. HTTP / API 拨测脚本

http_api_test.py Python 3 
#!/usr/bin/env python3
"""
HTTP / API 拨测脚本
输出: JSON 格式的 TTFB、TTLB、状态码、响应体哈希
用法: python3 http_api_test.py <url> [--method GET|POST] [--headers ...] [--body ...] [--timeout 30]
"""

import argparse
import hashlib
import json
import ssl
import time
from datetime import datetime, timezone
from http.client import HTTPSConnection, HTTPConnection
from urllib.parse import urlparse


def probe_http(url: str, method: str = "GET", headers: dict = None,
               body: bytes = None, timeout: int = 30) -> dict:
    """执行 HTTP 拨测，返回结构化结果"""
    parsed = urlparse(url)
    host = parsed.hostname
    port = parsed.port or (443 if parsed.scheme == "https" else 80)
    path = parsed.path or "/"
    if parsed.query:
        path = f"{path}?{parsed.query}"

    timestamp = datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")

    try:
        # 建立连接
        t_connect_start = time.monotonic()
        if parsed.scheme == "https":
            ctx = ssl.create_default_context()
            conn = HTTPSConnection(host, port, context=ctx, timeout=timeout)
        else:
            conn = HTTPConnection(host, port, timeout=timeout)

        # 发送请求
        req_headers = headers or {}
        if host and "Host" not in req_headers:
            req_headers["Host"] = host

        t_request_start = time.monotonic()
        conn.request(method, path, body=body, headers=req_headers)

        # 读取响应
        resp = conn.getresponse()
        t_ttfb = time.monotonic()

        resp_body = resp.read()
        t_ttlb = time.monotonic()

        conn.close()

        # 计算指标
        ttfb_ms = round((t_ttfb - t_request_start) * 1000, 2)
        ttlb_ms = round((t_ttlb - t_request_start) * 1000, 2)
        content_length = len(resp_body)
        body_hash = hashlib.sha256(resp_body).hexdigest()[:16]

        # 下载速率
        download_time = t_ttlb - t_ttfb
        download_rate_mbps = 0
        if download_time > 0:
            download_rate_mbps = round(
                (content_length * 8) / (download_time * 1_000_000), 2
            )

        result = {
            "test_type": "http_api",
            "target": url,
            "timestamp": timestamp,
            "params": {
                "method": method,
                "timeout_sec": timeout,
            },
            "result": {
                "status_code": resp.status,
                "status_reason": resp.reason,
                "ttfb_ms": ttfb_ms,
                "ttlb_ms": ttlb_ms,
                "content_length": content_length,
                "body_sha256_prefix": body_hash,
                "download_rate_mbps": download_rate_mbps,
                "response_headers": dict(resp.getheaders()),
            },
            "status": "ok" if 200 <= resp.status < 300 else "error",
        }

    except Exception as e:
        result = {
            "test_type": "http_api",
            "target": url,
            "timestamp": timestamp,
            "result": {
                "error": str(e),
                "error_type": type(e).__name__,
            },
            "status": "fail",
        }

    return result

def probe_openai_streaming(base_url: str, api_key: str, model: str = "gpt-4o-mini",
                           prompt: str = "Say hello in 50 words", timeout: int = 60) -> dict:
    """测试 SSE 流式响应"""
    import http.client
    import ssl as ssl_mod

    parsed = urlparse(base_url)
    host = parsed.hostname
    port = parsed.port or 443
    path = f"/v1/chat/completions"

    timestamp = datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")

    body = json.dumps({
        "model": model,
        "messages": [{"role": "user", "content": prompt}],
        "stream": True,
    }).encode()

    headers = {
        "Content-Type": "application/json",
        "Authorization": f"Bearer {api_key}",
        "Accept": "text/event-stream",
    }

    try:
        ctx = ssl_mod.create_default_context()
        conn = HTTPSConnection(host, port, context=ctx, timeout=timeout)
        conn.request("POST", path, body=body, headers=headers)

        resp = conn.getresponse()

        ttft_ms = None
        token_times = []
        token_count = 0
        done_received = False
        t_start = time.monotonic()

        while True:
            line = resp.readline()
            if not line:
                break

            line_str = line.decode("utf-8", errors="replace").strip()

            if line_str.startswith("data: ") and line_str != "data: [DONE]":
                token_count += 1
                now = time.monotonic()
                if ttft_ms is None:
                    ttft_ms = round((now - t_start) * 1000, 2)
                token_times.append(round((now - t_start) * 1000, 2))

            elif line_str == "data: [DONE]":
                done_received = True
                break

        t_end = time.monotonic()
        total_ms = round((t_end - t_start) * 1000, 2)

        # Token 间隔
        intervals = []
        for i in range(1, len(token_times)):
            intervals.append(round(token_times[i] - token_times[i-1], 2))

        avg_interval = round(sum(intervals) / len(intervals), 2) if intervals else None
        tps = round(token_count / ((t_end - t_start)), 2) if (t_end - t_start) > 0 else 0

        conn.close()

        result = {
            "test_type": "http_api",
            "test_subtype": "openai_streaming",
            "target": f"{base_url}/v1/chat/completions",
            "timestamp": timestamp,
            "params": {"model": model, "stream": True, "timeout_sec": timeout},
            "result": {
                "status_code": resp.status,
                "ttft_ms": ttft_ms,
                "total_ms": total_ms,
                "token_count": token_count,
                "tokens_per_second": tps,
                "avg_token_interval_ms": avg_interval,
                "done_received": done_received,
                "stream_interrupted": not done_received,
            },
            "status": "ok" if done_received else "warn",
        }

    except Exception as e:
        result = {
            "test_type": "http_api",
            "test_subtype": "openai_streaming",
            "target": f"{base_url}/v1/chat/completions",
            "timestamp": timestamp,
            "result": {"error": str(e), "error_type": type(e).__name__},
            "status": "fail",
        }

    return result

"""

验证 AI API 代理的业务逻辑正确性
用法: python3 business_logic_test.py --base-url https://api.example.com --api-key sk-xxx

import argparse
import json
import sys
import time
from datetime import datetime, timezone
from urllib.request import Request, urlopen
from urllib.error import HTTPError, URLError


class BusinessProber:
    """业务正确性拨测"""

    def __init__(self, base_url: str, api_key: str, timeout: int = 30):
        self.base_url = base_url.rstrip("/")
        self.api_key = api_key
        self.timeout = timeout
        self.timestamp = datetime.now(timezone.utc).strftime("%Y-%m-%dT%H:%M:%SZ")
        self.results = []

    def _request(self, method: str, path: str, body: dict = None,
                 headers: dict = None, expect_status: int = 200) -> dict:
        """执行 HTTP 请求"""
        url = f"{self.base_url}{path}"
        req_headers = headers or {}
        req_headers.setdefault("Authorization", f"Bearer {self.api_key}")

        if body:
            req_headers.setdefault("Content-Type", "application/json")
            data = json.dumps(body).encode()
        else:
            data = None

        req = Request(url, data=data, headers=req_headers, method=method)

        start = time.monotonic()
        try:
            resp = urlopen(req, timeout=self.timeout)
            ttfb = time.monotonic() - start
            resp_body = json.loads(resp.read())
            status = resp.status
            error = None
        except HTTPError as e:
            ttfb = time.monotonic() - start
            try:
                resp_body = json.loads(e.read())
            except Exception:
                resp_body = {"raw": str(e)}
            status = e.code
            error = str(e)
        except Exception as e:
            ttfb = time.monotonic() - start
            resp_body = {}
            status = 0
            error = str(e)

        return {
            "status_code": status,
            "ttfb_sec": round(ttfb, 3),
            "body": resp_body,
            "error": error,
            "pass": status == expect_status,
        }
